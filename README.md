# 🐍 Java Snake Game
<img width="611" height="633" alt="snakegame" src="https://github.com/user-attachments/assets/076239a1-a4cd-42d6-8798-68dd739149dc" />

A classic desktop Snake game built using Java and Swing/AWT. Navigate the snake around the screen, eat food to grow longer, and try to achieve the highest score without running into walls or your own tail!

---

## 🎮 Game Controls

| Action | Control Key |
| :--- | :--- |
| **Move Up** | `Up Arrow` / `W` |
| **Move Down** | `Down Arrow` / `S` |
| **Move Left** | `Left Arrow` / `A` |
| **Move Right** | `Right Arrow` / `D` |
| **Restart Game** | `Spacebar` / `R` (When Game Over) |

---
